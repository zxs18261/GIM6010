详细描述

应用入口模块（前后台调度）

注解
裸机前后台架构：
  • 前台：MC_FocIsr（ADC 注入完成中断，20kHz，见 motor_ctrl.c）、 CAN 接收解析（FDCAN ISR）、串口收发（USART/DMA ISR）；
  • 后台：APP_Loop 超级循环，非阻塞轮询 shell / 灯语 / 温度保护 / 周期日志 / 参数落盘，均以 HAL tick 分频，无任何忙等。

APP_Init 由 main.c 的 USER CODE BEGIN 2 调用（MX_*_Init 之后）， APP_Loop 由 while(1) 内调用。
