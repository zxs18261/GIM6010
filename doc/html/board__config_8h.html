详细描述

板级硬件参数统一配置（GIM4310/4305 电机控制器 V0.0 板）

注解
本文件集中定义与电路板绑定的全部硬件参数与引脚/通道映射， 依据为网表 Netlist_Schematic1_2026-07-11.tel 与原理图 V1.0：
  • TIM1 CH1/CH2/CH3 = U/V/W 相（FD6288Q 同相驱动）；
  • 电流采样：U/V 两相低侧腿独立 5mR + 公共母线回流 5mR， 内部 OPAMP 差分放大 25 倍，偏置约半量程（上电实测补偿）；
  • MT6701 编码器为 GPIO 位带模拟 SSI（PB4=CSN/PB5=CLK/PB6=DO， 无硬件 SPI 复用可用）；
  • EN_120（PA15）高电平接入 CAN 120R 终端电阻（TS5A3159 IN=H 连 NO）。
