详细描述

ADC 采样 BSP（电流/母线电压注入组 + 温度规则组）

注解
采样架构：
  • 注入组由 TIM1 TRGO2（OC4，计数器峰值附近）硬件触发，20kHz： ADC1 注入序列 = 相电流(U) + 母线电流 + 母线电压， ADC2 注入序列 = 相电流(V)；两 ADC 同触发准同步采样；
  • ADC1 JEOS（注入序列转换完成）中断即电流环节拍， 换算物理量后回调控制层；
  • NTC 温度走规则组，后台软件触发轮询（毫秒级）；
  • 电流换算：I = (raw - offset) * LSB / (增益*采样电阻)， 偏置在上电封波状态下实测平均（运放偏置 + 1.65V 基准合并补偿）；
  • CubeMX 生成的 ADC 参数（软件触发/单通道）为占位值，本模块重配。

◆ BSP_ADC_ReadRegularOnce()

float BSP_ADC_ReadRegularOnce ( ADC_HandleTypeDef * hadc)
static

软件触发单次规则组转换（NTC 慢通道专用）

注解
- 规则序列/采样时间已在 BSP_ADC_Init 中一次性配好，此处只做 启动-轮询-取值；单次模式下序列结束 ADSTART 自动清零， HAL 状态自动收回 READY，可反复调用；
  • 严禁调用 HAL_ADC_Stop：本 HAL 版本会连同注入组一起停止并 关闭 ADC，导致硬件触发的电流环节拍永久丢失；
  • 注入组优先级高于规则组，触发冲突时硬件自动仲裁（规则转换 被注入序列打断后自动恢复），本函数仅在后台低频调用， 阻塞几微秒。
参数
hadcADC 句柄（ADC1 = MOS NTC，ADC2 = 电机 NTC）
返回值
12bit原始值；失败返回 0
