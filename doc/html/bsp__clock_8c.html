详细描述

时钟切换 BSP（HSI -> HSE 20MHz，SYSCLK 维持 170MHz）

注解
CubeMX 生成的 SystemClock_Config 使用 HSI16 作 PLL 源；板载晶振 实为 20MHz（O_IN/O_OUT），HSI ±1% 精度对 1Mbps CAN 位时序裕量 不足，上电后应尽早切换到 HSE： HSE 20MHz / PLLM 5 * PLLN 85 / PLLR 2 = 170MHz（与 HSI 配置同频， 外设分频参数无需变更）。 切换失败（晶振异常）时自动回退 HSI 配置并返回错误，由应用层 打印告警，系统仍可降级运行。

HAL_StatusTypeDef BSP_CLOCK_SwitchToHse ( void )

切换系统时钟 PLL 源为 HSE

注解
在外设初始化早期调用（切换前后 SYSCLK 均为 170MHz， 已初始化外设的分频寄存器保持有效）。
返回值
HAL_OK已切换；HAL_ERROR HSE 启动失败，已回退 HSI 配置
