◆ BSP_CLOCK_SwitchToHse()

HAL_StatusTypeDef BSP_CLOCK_SwitchToHse ( void )

切换系统时钟 PLL 源为 HSE

注解
在外设初始化早期调用（切换前后 SYSCLK 均为 170MHz， 已初始化外设的分频寄存器保持有效）。
返回值
HAL_OK已切换；HAL_ERROR HSE 启动失败，已回退 HSI 配置
