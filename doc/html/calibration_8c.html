详细描述

预定位校准模块（编码器方向/极对数/电角度偏置）

注解
在电流环 ISR 上下文内以状态机方式执行，全程使用 d 轴电流闭环 + 强制电角度（不依赖编码器校准状态），流程：
  1. RAMP ：d 轴电流从 0 缓升至 CAL_CURRENT_A，theta 强制 0；
  2. SETTLE ：静置，转子锁定到强制角，记录编码器基准；
  3. ROTATE ：强制角匀速正向旋转 CAL_ROTATE_EREV 个电角圈， 由编码器增量判定方向、实测极对数；
  4. OFFSET ：分点锁定（正向/反向各 CAL_OFFSET_POINTS/2 点）， 每点记录 电角(编码器换算) - 强制角 的差， 正反双向平均以抵消摩擦/齿槽引起的滞后；
  5. RAMPDOWN：电流缓降，写入校准结果。 方向反转的处理：ROTATE 结束若编码器反向计数，立即写入反转标志并 重新预置编码器，后续 OFFSET 阶段在新方向语义下测量。
