enum MC_ModeTypeDef

控制模式枚举

注解
数值用于 CAN/shell 上报，勿改动既有取值。
枚举值
MC_MODE_DISABLED 

封波（freewheel）

MC_MODE_OPENLOOP 

开环电压模式（调试）

MC_MODE_TORQUE 

转矩（q 轴电流）环

MC_MODE_SPEED 

转速环

MC_MODE_POSITION 

位置环（级联速度环）

MC_MODE_MIT 

MIT 阻抗混合控制

MC_MODE_CALIB 

预定位校准中

MC_MODE_FAULT 

故障锁存

◆ MC_RebaseEncoder()

void MC_RebaseEncoder ( void )

重建编码器多圈基准并保持用户位置连续（参数落盘后调用）

注解
落盘期间全程关中断约 20ms+，编码器停采；若期间关节被外力 反拖超过转子半圈，多圈跳变判定会滑圈。本函数重新预读编码器 并调整会话零位使 Pos 读数连续（假设落盘期间关节未动）。 须在 FOC 中断屏蔽状态下由后台调用。
返回值
None

◆ MC_SetOpenLoop()

void MC_SetOpenLoop ( float vd,
float vq,
float speedE )

设置开环电压矢量（OPENLOOP 模式）

注解
开环不经电流 PI，电压矢量幅值限制为 Rs*峰值电流对应的 稳态压降，防止误输入造成持续大电流（开环无温度降额通道， 贴轨检测与软件过流阈值仍然生效）。
参数
vd/vqd/q 轴电压 [V]
speedE电角速度 [rad/s]（内部积分生成旋转角）
返回值
None
