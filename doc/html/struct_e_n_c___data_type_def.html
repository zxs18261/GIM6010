编码器运行数据结构体

注解
多圈信息保持在整数域（Turns int32，±21 亿圈不溢出）， 不提供合成的连续计数（int32 合成计数在高速下约 1 小时即溢出， 浮点合成在大圈数下损失单 count 分辨率）——由使用方按 Turns 与 Raw 分离计算。
