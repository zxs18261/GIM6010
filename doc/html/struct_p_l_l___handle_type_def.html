详细描述

PLL 型角度跟踪速度估计器句柄

注解
以锁相环结构跟踪输入的单圈角度（回卷感知相位差），输出平滑角速度； 内部跟踪角保持在 [0, 2pi) 回卷域，长时间连续旋转无精度退化/溢出 （相比跟踪连续展开角，float 大数相减的灾难性抵消被规避）。 相比直接差分对编码器量化噪声抑制显著。
